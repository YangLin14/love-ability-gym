## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/love-ability-gym.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `npm install`
5. Start development server: `npm run dev`

## Development Guidelines

### Code Style

- Use meaningful variable and function names
- Write components in functional React style with hooks
- Keep components small and focused
- Use the existing translation system for all user-facing text

### Commit Messages

Use clear, descriptive commit messages:
- `feat: add new emotion tracking feature`
- `fix: resolve chart rendering issue`
- `docs: update README with new instructions`

### Translations

## Submitting Changes

1. Ensure your code runs without errors
2. Test on mobile viewport sizes
3. Push to your fork
4. Open a Pull Request with a clear description
